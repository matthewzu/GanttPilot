# 更新日志 / Changelog

## [1.3.1] - 2026-05-07

### 改进 / Improved
- 👤 **项目级提交者配置** - 每个项目可以设置独立的提交者名称和邮箱，留空则自动使用全局配置
- 📝 **提交者字段移至项目编辑** - 提交者信息从 Git 配置对话框移到项目编辑对话框，本地模式和协作模式创建项目时均可填写
- 💡 **全局 Fallback 提示** - 项目编辑对话框中提交者字段留空时显示"将使用全局配置: xxx <xxx@xxx>"提示
- 🔒 **提交者信息不入库** - 提交者信息保存在本地全局配置的 `project_committers` 字典中（按项目名索引），不会被提交到 git 仓库

## [1.3.0] - 2026-05-07

### 新增 / Added
- 🔗 **更新对话框可点击链接** - 软件更新前弹出对话框包含可点击的 CHANGELOG 链接，更新完成后包含可点击的 README 链接
- 🔄 **更新后自动重启** - 下载更新完成后自动重启应用，无需手动操作
- ⚙️ **提交者信息全局化** - 提交者名称和邮箱从项目级别移至全局配置，所有项目共享，避免 PR 合并后信息不一致导致拉取失败
- 🧹 **自动迁移** - 加载项目时自动清除 project.json 中的旧提交者字段，下次同步即生效
- 🔀 **分支名安全生成** - 自动生成私有分支名时将空格等非法字符替换为下划线（如 "Kevin Fang" → `priv_Kevin_Fang`）
- 🔀 **旧分支自动重命名** - 如果本地存在旧的无效分支名（含空格），init_repo 时自动重命名为合法名称

### 修复 / Fixed
- 🐛 **PR 合并后拉取失败** - 修复提交者名称含空格时生成无效分支名（如 `priv_Kevin Fang`）导致 git pathspec 错误的问题
- 🐛 **Git 配置对话框** - 移除项目级提交者字段，简化为仅配置远端和私有分支名

### 向后兼容说明
- 已有项目中的 `committer_name`/`committer_email` 字段会在加载时自动清除并写回文件
- 全局配置（`~/.ganttpilot/config.json`）新增 `committer_name` 和 `committer_email` 字段
- 首次使用协作模式创建项目时输入的提交者信息会自动保存到全局配置

## [1.2.0] - 2026-05-07

### 新增 / Added
- 🔄 **手动更新检测** - 工具栏新增 ⟳ 按钮，手动检查是否有新版本可用，检测期间按钮自动禁用防止重复点击
- 📝 **活动描述字段** - 活动新增多行描述输入框，与需求描述具有相同的编辑特性（带滚动条），向后兼容旧数据
- 🔀 **fetch 后自动更新 main** - 拉取/同步后自动将本地 main 分支快进到 origin/main，无需手动切换分支执行 pull
- 💬 **rebase 提示** - main 分支有更新时自动弹窗询问是否将私有分支变基到最新 main，冲突时自动中止并提示
- 🚫 **禁止向 main 推送** - sync 操作仅推送私有分支，不会向远端 main 分支推送
- ✅ **私有分支名校验** - 创建/编辑项目时校验私有分支名称，禁止输入 "priv"、与主分支同名或含非法字符

### 修复 / Fixed
- 📐 **对话框居中** - 所有模态对话框（添加项目、编辑计划等 15 个）改为居中于主窗口显示，修复从 (0,0) 开始导致被任务栏遮挡的问题
- 🔧 **clone_repo 分支名修复** - clone_repo 使用用户配置的 priv_branch 名称创建私有分支，而非硬编码 "priv"

### 向后兼容说明
- 已有项目中使用 "priv" 分支的配置继续正常工作，不强制迁移
- 旧活动数据加载时自动填充空描述字段

## [1.1.0] - 2026-05-06

### 新增 / Added
- 🆕 **项目创建双模式** - "添加项目"对话框拆分为本地模式和协作模式，通过单选按钮切换
- 🏠 **本地模式** - 仅需填写项目名称和描述（可选），快速创建本地项目
- 🤝 **协作模式** - 一步完成 Git 配置（远端地址、分支、用户名、密码、提交者信息、私有分支），无需单独打开 Git 配置对话框
- 💡 **输入提示** - 所有输入框添加占位提示信息（PlaceholderEntry），引导用户正确填写
- ✅ **字段验证** - 协作模式下验证远端仓库地址格式，提交者信息为空时自动检测系统 Git 配置
- 📐 **对话框自适应** - 切换模式时对话框自动调整尺寸

## [0.11.0] - 2026-04-23
