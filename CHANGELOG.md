# 更新日志 / Changelog

## [0.11.0] - 2026-04-23

### 新增 / Added
- ⏪ **回退到此处** - 历史记录标签页右键提交记录，选择"回退到此处"执行 `git reset --hard`，丢弃之后所有提交
- ↩️ **撤销此提交** - 历史记录标签页右键提交记录，选择"撤销此提交"执行 `git revert`，生成反向提交保留历史

### 改进 / Improved
- 🔄 **同步策略优化** - 启动时仅自动 fetch（拉取远端更新），不再自动 push；切换项目和关闭窗口时不再自动同步；推送操作仅通过手动同步（右键菜单/Ctrl+S）触发

### 修复 / Fixed
- 🐍 **Python 兼容性** - 修复 `str | None` 类型注解在 Python 3.9 以下版本导致启动崩溃的问题，添加 `from __future__ import annotations`

## [0.10.0] - 2026-04-22

### 新增 / Added
- ⌨️ **键盘快捷键系统** - 新增 ShortcutManager 集中管理快捷键：Ctrl+N 添加、F2 编辑、Delete 删除、Alt+Up/Down 上移/下移、Ctrl+D 克隆、Ctrl+S 同步、F5 刷新
- ⚙️ **快捷键配置** - 配置对话框新增快捷键配置区域，支持自定义快捷键绑定、冲突检测、恢复默认
- 💡 **快捷键提示** - 工具栏按钮 tooltip 和右键菜单显示对应快捷键信息
- ⎋ **ESC 关闭对话框** - 所有 15 个对话框支持 ESC 键快速关闭

### 改进 / Improved
- 🔤 **字体缩放增强** - 通过 TkDefaultFont/TkTextFont 全局字体缩放，所有对话框控件（Entry、Label、Text）跟随缩放
- 📐 **对话框布局优化** - 所有对话框添加 columnconfigure 和 sticky=EW，拉伸时输入框自动扩展
- 🗑️ **配置精简** - 全局配置对话框移除 Git 相关字段（远端地址、用户名、密码），Git 配置仅在项目级别管理

### 修复 / Fixed
- 🪟 **对话框焦点恢复** - 修复打开对话框后 Alt-Tab 切换窗口再切回时无法操作的问题，添加 FocusIn 焦点恢复机制和 _active_dialog 跟踪
- ✏️ **描述框撤销/恢复** - 修复 tk.Text 描述框中 Ctrl+Z/Y 触发全局 UndoManager 而非文本撤销的问题，为所有描述框启用 undo=True 并添加焦点检测
- ↕️ **移动快捷键修复** - 修复任务/需求节点移动快捷键有时不生效的问题，移动后立即更新工具栏按钮状态
- 📅 **活动日期可选** - 活动对话框中日期字段改为可选项，标签显示"可选"提示

## [0.9.1] - 2026-04-21

### 新增 / Added
- ⏱ **工时（小时数）** - 活动新增"工时（小时数）"字段，允许直接输入小时数，与"工时（时间段）"二选一填写，互斥校验并提供明确提示
- 📊 **项目总工时** - 项目报告新增"项目总工时"章节，显示项目合计工时及每位执行者的工时与占比
- 📊 **分组工时合计** - 工时统计界面所有查看模式（按项目/里程碑/计划/标签）均显示每个分组的总工时行
- 📊 **工时占比** - 工时统计界面和项目报告中显示每位执行者工时占当前分组总工时的百分比

### 改进 / Improved
- 🏷️ **工时时段更名** - "工时时段"统一更名为"工时（时间段）"，"工时(小时数)"统一为"工时（小时数）"
- 🌳 **活动树形显示** - 活动在树形图中直接显示小时数（如 `3.0h`），不再显示时间段字符串
- 📝 **示例项目更新** - 示例项目描述和活动内容更新，体现"直接填写小时数"的新用法

## [0.9.0] - 2026-04-21

### 新增 / Added
- ⧉ **节点克隆** - 一键克隆任意节点（项目、需求、任务、里程碑、计划、活动）及其所有子节点，自动重新生成 ID 并添加 "(Copy)" 后缀
- 📋 **复制粘贴** - 复制节点到内存剪贴板，支持跨项目/跨里程碑/跨需求粘贴，自动校验目标位置类型兼容性
- 🛠️ **工具栏按钮** - 新增复制(📋)、粘贴(📌)、克隆(⧉) 三个工具栏按钮，根据选中节点类型和剪贴板内容自动启用/禁用
- ⌨️ **快捷键** - Ctrl+C 复制、Ctrl+V 粘贴
- 📝 **右键菜单** - 所有节点类型的右键菜单增加复制/粘贴/克隆选项，粘贴选项仅在类型兼容时显示

### 改进 / Improved
- 🛠️ **工具栏布局优化** - 按钮按逻辑分组：增/编辑/删除 → 复制/粘贴/克隆 → 上移/下移
- 📋 **右键菜单排序优化** - 统一排序：添加子节点 → 编辑/属性 → 剪贴板操作 → 状态操作 → 工具操作 → 删除

### 修复 / Fixed
- 🧪 修复 hypothesis 属性测试因默认 200ms deadline 导致的间歇性失败，为所有涉及文件 I/O 的测试设置 `deadline=None`

## [0.8.1] - 2026-04-20

### 新增 / Added

- ✅ **任务管理** - 将需求拆解为任务，支持工作量估算（人日）
- 🔗 **需求跟踪** - 计划关联任务，需求跟踪标签页查看完整跟踪链条
- 🛠️ **统一工具栏** - 添加、编辑、删除、上移、下移按钮
- 📝 **报告增强** - Markdown 报告包含需求分析和需求跟踪章节
